# 🌍 Calculadora d'Estalvi Energètic

Una aplicació web moderna per a l'anàlisis de consumos energètics, d'aigua, consumibles d'oficina i productes de neteja amb projeccions futurs i sugerències d'estalvi basades en dades.

**🔗 Accés en línia:** https://itb2526-adrianflecha.github.io/Calculadora-estalvi-energetic/

## ✨ Característiques

- 📊 **Dashboard Interactiu**: Visualització de métriques principals amb gràfics en temps real
- 📝 **Entrada de Dades Manual**: Afegir, editar i eliminar registres de consumo
- 📈 **Análisis Detallat**: Comparatives entre períodes escolars i l'any complet
- 🔮 **Projeccions Futures**: Cálculs amb inflació del 3% acumulativa
- 🌱 **Sugerències d'Estalvi**: Recomanacions automàtiques basades en les teves dades
- 📅 **Períodes Especials**: Análisis de Setmana Santa, Estiu i Nadal
- 📥 **Exportació a HTML**: Descàrrega informes en format HTML
- 💾 **Persistència Local**: Emmagatzemament de dades al navegador (localStorage)
- 🌐 **100% Frontend**: Cap servidor, tot es processa al navegador
- 📱 **Responsive Design**: Funciona perfectament en móbil, tablet i desktop
